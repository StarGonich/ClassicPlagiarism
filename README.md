Предустановка необходимых программ
- Для тестирования использовался Python 3.12, хотя может и на других работать
- Установка Joern 4.0.xxx версии, к примеру стабильно работает версия 4.0.426 [(скачать)](https://github.com/joernio/joern/releases/download/v4.0.426/joern-cli.zip). Далее для Windows добавить в переменные пути папку .../joern-cli/bin
- Для ускорения работы программы рекомендуется установить PyPy 3.11 [(скачать для Windows 64bit)](https://downloads.python.org/pypy/pypy3.11-v7.3.23-win64.zip)

Использование
3) Запуск sh файла, где в качестве параметра используется директория с программами
```bash
.\files_to_graph.sh .\-1\
```
В итоге появится несколько папок, но нас в основном интересует test_graphs, где находятся PDG графы наших программ для теста.

4) Запуск main.py, где берутся графы из директории `test_graphs` и сравниваются. По итогу создаётся директория `test_logs`, где хранится выводы
5) Запуск gui.py, где можно в графическом интерфейсе посмотреть на результаты.
